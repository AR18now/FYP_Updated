Project: AGIT - Automated Game Genre Identification Using Text and Image Classification

1. INTRODUCTION
1.1 Purpose
This document outlines the requirements for AGIT, a system that automates game genre identification by integrating both textual (game summaries) and visual (gameplay images) classification using advanced deep learning techniques and transformers.

1.2 Scope
The system aims to classify games into 23 diverse genres using a multimodal approach. It will leverage the comprehensive MSR PlayMyDataset (containing game summaries and images) and employ transformers (BERT, RoBERTa) for text analysis and CNNs/YOLO for image analysis. The goal is to create a robust multilabel classification model.

1.3 Definitions, Acronyms, Abbreviations
- NLP: Natural Language Processing
- CNN: Convolutional Neural Network
- LLM: Large Language Model
- SVM: Support Vector Machine

2. OVERALL DESCRIPTION
2.1 Product Perspective
AGIT addresses the limitations of previous studies that focused on single-modality classification or a limited set of genres by combining text and image analysis to provide a more comprehensive and accurate genre identification model.

2.2 Product Functions (Summary)
- User-friendly web portal.
- Integration of gameplay images and textual summaries for classification.
- Deep learning and transformer-based approaches for multilabel pipelines across 23 genres.
- Input field for game summaries.
- Classification scores and criteria for authentic result evaluation.
