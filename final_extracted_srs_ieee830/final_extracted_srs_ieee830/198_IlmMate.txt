Project: IlmMate - WhatsApp-Based Educational Chatbot for Pakistani Students
================================================================================

1. INTRODUCTION
1.1 Purpose
This document outlines the requirements for IlmMate, a WhatsApp-based educational chatbot designed to empower Pakistani grade 9 and 10 students by providing curriculum-specific explanations, e-learning resources, and AI-generated quizzes to facilitate learning at their own pace and convenience.

1.2 Scope
The system uses OCR to digitize textbooks and stores content in vector databases. It processes user queries via WhatsApp API, shortens queries, retrieves relevant content, and uses an LLM to generate responses. Features include automated quiz generation, e-learning resource retrieval, progress tracking, and report generation. It aims to address educational inequality.

1.3 Definitions, Acronyms, Abbreviations
- OCR: Optical Character Recognition
- LLM: Large Language Model
- API: Application Programming Interface
- RAG: Retrieval-Augmented Generation

2. OVERALL DESCRIPTION
2.1 Product Perspective
IlmMate leverages the widespread use of WhatsApp in Pakistan to provide an accessible, platform-independent educational tool that helps students learn curriculum topics with detailed explanations and self-assessment, especially in under-served areas.

2.2 Product Functions (Summary)
- Backend Module: Digitize textbooks (OCR), store in vector databases, shorten user queries, retrieve relevant content, generate responses via LLM, track student performance.
- WhatsApp Chatbot: Automated assessment (generate questions), e-learning resource agent (retrieve video links), communicate via WhatsApp API.
- System Integration and Testing: Integrate components, set up guardrails, conduct testing.

2.3 User Classes and Characteristics
- Student: Main target user (grades 9-10) accessing curriculum content, quizzes, resources.
- Parent: Receives performance reports, monitors child's progress.
- Teacher: Tracks progress of students in their classroom.
- Educational Institution: Integrates IlmMate into existing curricula.

3. FUNCTIONAL REQUIREMENTS
3.1 Backend Module: Content Management and Retrieval
FR1: The system should use OCR to convert textbooks into digital text and store it in a vector database.
FR2: The system must shorten user queries to focus on their main meaning.
FR3: The system should find and retrieve relevant content from the vector database.
FR4: The system should track student performance and generate progress reports.

3.2 WhatsApp Chatbot
FR1: The system should communicate with users through the WhatsApp API.
FR2: The system should automatically generate practice questions based on topics chosen by users.
FR3: The system must provide e-learning resources, such as video links, based on user requests.

3.3 System Integration and Testing
FR1: The system must integrate all parts of IlmMate.
FR2: The system must have safeguards to manage off-topic queries and reduce incorrect responses.
FR3: The system must perform testing to ensure everything works correctly.

4. NON-FUNCTIONAL REQUIREMENTS
4.1 Reliability
NFR1: The system must achieve a Mean Time Between Failures (MTBF) of at least 48 hours.
NFR2: In case of failure, the system should implement automatic recovery mechanisms, sending alerts to developers.

4.2 Usability
NFR3: The chatbot interface will be intuitive and simple to navigate.
NFR4: The interface will provide clear feedback during response generation using filler sentences (e.g., "Let me get the answer to that for you.").

4.3 Performance
NFR5: The system must respond to user queries within 20 seconds, even during high traffic.
NFR6: The system will be optimized to support up to 50 concurrent users without performance degradation.
NFR7: The system aims for a 90% accuracy rate in LLM-generated responses, with continuous fine-tuning.

4.4 Security & Privacy
NFR8: Only authenticated users will have access to the system's features.
NFR9: The system could keep parents in sync with their child's performance, usage, and chat logs if possible.
