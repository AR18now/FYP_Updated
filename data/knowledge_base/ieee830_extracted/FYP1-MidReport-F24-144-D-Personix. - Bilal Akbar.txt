Chapter 3
System Overview
Personix is an Android application focused on improving digital self-representation through
3D facial models. Users can upload photographs, which are processed using computer
vision and facial recognition algorithms to generate detailed 3D avatars. These avatars
can be customized to match the user’s preferences, allowing changes to features such as
head and facial hair, and facial structure. Additionally, Personix integrates text-to-speech
technology with sentiment analysis, enabling the avatars to speak and exhibit appropriate
facial expressions. The platform also supports media creation, including GIFs and videos,
and offers cloud storage for cross-device synchronization.

3.1.2 Modular Program Structure
The system is designed in a modular manner, where each module plays a distinct role in
achieving the overall functionality. Below is a brief explanation of the major modules and
their relationships:
• User Input: Collects user data such as images and text, which is passed to other
processing components.
• Image Processing: Processes the image input for validation and then use in 3D
Model Generation.
• 3D Model Generation : Based on the processed images, this module generates
a 3D avatar. The generated model is stored in the Data Store for further use or
customization.
• Text to Speech: Converts user-provided text into speech and passes the resulting
speech to the Sentiment Analysis module.
• Sentiment Analysis: Analyzes the emotional tone of the speech input and sends
the results to Facial Expression Mapping.
• Facial Expression Mapping: Maps the corresponding facial expressions onto the
3D avatar based on the sentiment analysis.

3. System Overview
• Avatar Customization: Allows users to modify and customize their 3D avatars.
These modifications are stored back in the Data Store.
• Data Store: Centralized storage for the 3D models and other data. It is accessed by
both Avatar Customization and Video Rendering/Exporting.
• Video Rendering/Exporting: Generates videos based on the customized avatars
and provides options for exporting or sharing the content.
Some of the modules operate independently, but together they contribute to the system’s
goal of creating, customizing, and rendering expressive avatars.
