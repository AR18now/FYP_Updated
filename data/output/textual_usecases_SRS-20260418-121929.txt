Use Case: FR-01 - Route Planning
Primary Actor:Passenger
Stakeholders:Operations Team
Precondition:Passenger has entered origin and destination into the application.
Main Success Scenario:1. Application calculates and displays the most efficient route based on predefined criteria.
2. Passenger reviews and selects the preferred route.
Extensions:- If no suitable route exists, the application suggests alternative options.
Postcondition:Passenger knows which route to follow.
Trace to FR Id:FR-01
Use Case: FR-02 - Ticket Purchase
Primary Actor:Passenger
Stakeholders:Operations Team
Precondition:Passenger has selected a route and chosen their passenger type.
Main Success Scenario:1. Application generates a ticket with the correct fare and concession discounts applied.
2. Passenger receives a confirmation message and ticket details.
Extensions:- If the passenger does not qualify for a discount, they receive a standard ticket.
Postcondition:Passenger holds a valid ticket.
Trace to FR Id:FR-02
Use Case: FR-03 - Single Ticket Validation
Primary Actor:Passenger
Stakeholders:Conductors
Precondition:Passenger presents a single-use ticket at the entry point.
Main Success Scenario:1. Entry point machine scans the ticket.
2. Machine verifies the ticket’s validity and applies penalties if necessary.
3. Machine displays a green light and allows passage.
Extensions:- If the ticket is invalid, the machine denies passage and provides feedback.
Postcondition:Passenger either enters the vehicle or is informed about the issue.
Trace to FR Id:FR-03
Use Case: FR-04 - Monthly Pass Validation
Primary Actor:Passenger
Stakeholders:Conductors
Precondition:Passenger presents a monthly pass at the entry point.
Main Success Scenario:1. Entry point machine scans the pass.
2. Machine verifies the pass’s validity and updates the ride count.
3. Machine displays a green light and allows passage.
Extensions:- If the pass is expired or invalid, the machine denies passage and provides feedback.
Postcondition:Passenger either enters the vehicle or is informed about the issue.
