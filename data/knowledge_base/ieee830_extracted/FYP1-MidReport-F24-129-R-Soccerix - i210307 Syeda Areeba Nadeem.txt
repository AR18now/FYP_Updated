1. Introduction
1.2 Research Problem Statement
The lack of an automated, impartial, and effective system for tactical football analysis
is the research challenge. Conventional approaches, which rely on physical labour and
human interpretation, are not consistent enough to provide a complete picture of team
performance. Soccer’s increasing focus on analytics has made an AI-driven solution more
urgently necessary. Coaches and analysts could be able to make more educated tactical
decisions with the help of such a system, which could offer precise, comprehensive post-
match analysis. By examining how AI can provide more dependable, comprehensive, and
timely insights on the tactical intricacies of the game, this research seeks to close the gap.

2.1.2 Tracking of Multi-Athlete and Action Recognition in Soccer
Sports Video Using Deep Learning Techniques
2.1.2.1 Summary
The paper [55] focuses on automating the detection and recognition of key events in soc-
cer games using deep learning models. The methodology includes pre-processing soc-
cer video frames, extracting features using deep learning models like ResNet152, Naive
Bayes, and SVM, and classifying events such as goals, tackles, and fouls. TheSoccerNet-
v2 dataset, consisting of 550 soccer videos from top European leagues, was used to
train and test the models. The proposed system is structured into data collection, pre-
processing, feature extraction, and event prediction, achieving an accuracy of 88% with
the ResNet152 model. The paper aims to reduce the time-consuming manual annotation
process by automating event detection.
2.1.2.2 Critical Analysis
The paper’s strengths include the successful integration of multiple deep learning models,
which improved action recognition accuracy, especially using concatenated pre-computed
features. This approach enhanced performance, as seen in the ResNet152 model’s 88%
accuracy. However, the reliance on precomputed features rather than raw video data lim-
its flexibility and scalability. Additionally, the system’s dependence on a single dataset
(SoccerNet-v2) could lead to overfitting, and the lack of real-time implementation testing
raises concerns about its practical applicability.

2.1.3 Soccer Video Event Detection Using Metric Learning
2.1.3.1 Summary
The paper [ 33] proposes a solution for detecting events in soccer videos using convolu-
tional neural networks (CNNs) and gated recurrent units (GRUs). This makes use of
Siamese networks that uses contrastive loss in detecting different soccer events such as
goals, fouls, penalty and corner kicks. In this work we take as reference the SoccerEvent
and SoccerNet datasets. The approach attains 64.9% mAP on SoccerNet and 93.76%
mAP in the SoccerEvent data set. The best-fit architecture in our case was EfficientNetB0
and GRU.This paper demonstrates the effectiveness of metric learning for event detec-
tion, showing that model trained by pair-wise ranking can learn a more general feature
and perform cross-match generalization over diverse soccer game data.
2.1.3.2 Critical Analysis
This paper is to fully utilize the spatial and temporal information from soccer videos by
using Siamese networks within a deep learning architecture for CNN and GRU. More-
over, by applying contrastive loss, the event detection accuracy can be further boosted to
93.76% mAP on SoccerEvent dataset. In addition, collecting a diverse dataset also helps
the model to generalize better across different conditions such as stadiums, teams etc. De-
spite this, the model records a relatively lower performance (64.9%) on SoccerNet dataset
that could be an indication of shortfalls in dealing with certain events or datasets with less
diversity. Furthermore, the system’s reliance on computationally intensive architectures
like EfficientNetB0 may limit its scalability and real-time application. Expanding the
evaluation across more diverse soccer datasets or exploring lightweight architectures for
real-time analysis could address these gaps.

2.1.4 RMS-Net: Regression and Masking for Soccer Event Spotting
2.1.4.1 Summary
The paper [88] introduces a lightweight network for action spotting in soccer videos, fo-
cusing on detecting key events such as goals, substitutions, and cards. The methodology
involves predicting both the event class and its temporal offset using 2D convolutional
features with a masking strategy to handle ambiguous frames. The system is tested
on the SoccerNet dataset , containing 500 full soccer matches, and achieves a 65.5%
Average-mAP on the test set, surpassing prior methods. The model integrates temporal
convolutions and regression strategies to improve temporal precision. Two techniques,
data balancing and masking of irrelevant frames, are incorporated to mitigate class im-
balance and enhance feature extraction.
2.1.4.2 Critical Analysis
RMS-Net excels in its lightweight design and modularity, allowing for easy integration
with existing models while addressing event sparsity in soccer videos. Its innovative use
of temporal regression improves the precision of event spotting, outperforming prior base-
lines. However, the model struggles with detecting subtle events like cards due to limited
visual cues, and the masking strategy, while effective, may reduce context, leading to mis-
classifications in complex scenarios. Moreover, the system heavily relies on pre-extracted
features, limiting its adaptability to more dynamic contexts like player interactions. Fur-
ther improvements could focus on enhancing recognition of minor events and reducing
reliance on specific pre-trained models.
2.1.4.3 Relationship to Soccerix
The paper’s approach to detecting key soccer events using temporal features and mask-
ing aligns with Soccerix’s aim to capture and analyze events like goals and free kicks.
The method’s emphasis on accuracy and event detection efficiency directly supports our
system’s goal of providing data-driven insights to improve team tactics and gameplay
decisions.

2.1.5 Extraction of Positional Player Data from Broadcast Soccer
Videos
2.1.5.1 Summary
The paper [ 77] presents a modular pipeline for fully-automated extraction of positional
player data from broadcast soccer videos. The system includes several key components:
sports field registration, player detection, team assignment, and position estimation. The
approach leverages homography matrices to transform player positions from image to
field coordinates. The system was evaluated on multiple datasets, including broadcast
videos from the German Bundesliga, and achieved promising results, with the mean po-
sition error being less than 2 meters for most frames. Models like Faster R-CNN and
CenterTrack are employed for player detection. The pipeline’s modularity enables indi-
vidual evaluation of each sub-task, with the ultimate goal of assisting in tactical analysis
and player performance evaluation. Player detection achieved average precision of 92.8%
on the Soccer Player dataset and 90.1% on the ISSIA-CNR dataset.
2.1.5.2 Critical Analysis
The pipeline’s strengths include its generalizability to new datasets, modular design, and
robust evaluation across different camera settings. The use of homography matrices al-
lows accurate mapping of player positions from video frames to field coordinates. How-
ever, the system’s modular approach, while beneficial for flexibility, may compound errors
between sub-tasks, reducing overall accuracy. Additionally, the system’s reliance on vi-
sual input makes it vulnerable to inaccuracies caused by poor video resolution or lighting.
Future work could focus on improving player tracking and addressing occlusion issues
for more consistent results.
2.1.5.3 Relationship to Soccerix
The paper’s pipeline for extracting player positions from broadcast videos, using models
like Faster R-CNN and homography matrices, aligns with Soccerix’s AI-based player and
ball tracking. It also supports key Soccerix goals like event detection and tactical analysis,
offering a solid foundation for data-driven insights in soccer.

The dataset used for Soccerix is sourced from SoccerNet [66]. For event detection, the
SoccerNet Action Spotting dataset includes 500 soccer broadcast videos and covers 17
event categories: Penalty, Kick-off, Goal, Substitution, Offside, Shots on target, Shots
off target, Clearance, Ball out of play, Throw-in, Foul, Indirect free-kick, Direct free-
kick, Corner, Yellow card, Red card, and Yellow->red card. For object detection and
tracking, the SoccerNet Tracking dataset includes classes such as player team left, player
team right, goalkeeper team left, goalkeeper team right, main referee, side referee, staff,
and ball, based on 100 video clips of 30 seconds each at 1080p resolution.
The Soccerix system begins by analyzing video footage, detecting key events like goals,
free kicks, and corner kicks while using object detection to identify players, ball, goal-
keepers and referees on the field. It tracks object movements, assigns players to teams,
and analyzes team formations. Soccer field key points are detected for creating a bird’s-
eye view of the field using perspective transformation, highlighting player positions. The
detected events are linked to specific teams, providing contextual insights. Finally, this
data is transformed into detailed analysis, automated reports, and visualizations, offering
coaches and analysts a comprehensive understanding of the match through dashboards
that present actionable metrics and performance evaluations.

3.0.6 Team Classification
Team classification refers to the process of grouping individuals into distinct teams based
on visual data. Various models can be used to generate embedding vectors for players
using contrastive learning, allowing the model to create meaningful representations of
players based on their visual features. If the learned representations are of significantly
higher dimensions, dimensionality reduction techniques can be employed. Models like
autoencoders combine these two steps into one. Afterwards, clustering can be performed
to classify the players into their respective teams.
3.0.6.1 Models for Representation Learning:
• Sigmoid Loss for Image and Language Pre-training (SigLIP): This model gen-
erates image embeddings using a contrastive learning framework. The vision model
from SigLIP, without any head or projection on top, generates image embeddings.
It focuses on learning robust representations from visual data without any text input.
• Siamese Networks: Used to learn embeddings for images by comparing pairs of
images and training the network to minimize the distance between similar images
and maximize the distance between dissimilar images.
