1.2 Scope
RastaSaaz is a web based solution designed to assist graduates in crafting optimized,
professional resumes tailored to the specific job descriptions. Our system will parse the
resume to extract all the relevant information. Once the resume is parsed its skills will be
matched with those required in job description, on the basis of skills matched the resumes
will be ranked. The resumes with high rank will be automatically refactored, rearranging
sections to highlight the most relevant content at the top and removing irrelevant details.
Our system narrowly focuses on IT resumes, neglecting to explore different domains with
distinct resume styles and technologies. The refactoring process will not include creating
projects and experience description by itself, but it would only arrange the project priority
wise according to the job description on LinkedIn. Moreover, our system will refine
the professional summary of the user to include specific keywords mentioned in the job
descriptions. After the refactoring process, our system will apply on multiple jobs using
automation.

System Overview
Rasta Saaz is an AI-driven Resume refactoring system for fresh graduates
and job seekers. Its primary objective is to automate the task of applying
to multiple IT related jobs by refactoring the resumes according to the job
descriptions on linkedin, with minimum human intervention.
Backgound
RastaSaaz emerges in response to the challenge of manually tailoring re-
sumes for each job application, a process that is both time-consuming and
labor-intensive. Despite the increasing demand for personalized resumes,
there is a notable lack of systems that automate this process. To address
this gap, our platform automates resume refactoring, enabling users to effi-
ciently adapt their resumes to meet specific job requirements without man-
ual intervention.
Context
The System is developed to cater the growing demand of the automatic
resume refactoring system. Our system will generate a personalized and
specific refactored resume which would enhance the chances of getting a
job.
Design
RastaSaaz design divides the application into distinct layers which are pre-
sentation, business logic, data layer. Each of these layers will focus on one

Functionality

3. System Overview
Multi Layer Architecture
A multi-layer architecture is ideal for RastaSaaz due to its structured sep-
aration of concerns. In this architecture, the presentation layer provides an
intuitive user interface for job seekers. It provides an intuitive user inter-
face for job seekers, facilitating seamless interactions with system func-
tionalities.A smooth user experience is delivered in the presentation layer,
which manages the user interface and interaction elements.
The Application layer encompasses essential modules such as job-seeker’s
profile management, resume parsing, job scraping, skill gap analysis, job
matching, resume ranking and resume refactoring. These modules will
contain the business logic required to make the function on the interface
operate smoothly.
The data layer efficiently handles database operations, storing user profiles,
resumes, job descriptions, and matching data, ensuring data integrity and
optimized access.
Additionally, while the API access component interfaces with external ser-
vices to enrich functionality—such as fetching job listings—it operates
outside the core layers, allowing seamless integration without complicat-
ing the architecture. Moreover, an audit log is crucial for tracking user
actions and changes, enhancing accountability and troubleshooting, while
security features protect sensitive user data and ensure compliance with
best practices, making the overall system robust and reliable.
